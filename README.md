# ⚙ Projeto

**moveit** é uma aplicação web para se exercitar utilizando em conjunto da técnica pomodoro. Foi desenvolvido durante a quarta edição do evento Next Level Week, da rocketseat. ❤️

# ✈ Tecnologias

- Next.js
- React
- TypeScript
- ContextAPI
